Kotlin Mumbai: TDD using Spek

Jul 1, 2020  ·  1 min read

Senior Android Engineer at BYJU’s, Rivu Chakraborty gives a talk on test driven development using Spek

Event details

Speaker: Rivu Chakraborty, BYJUs, Senior Android Engineer

Talk: TDD using Spek

Date: June 23rd, 2020

Location: Virtual

In case you missed it

What we covered

Rivu walks us through how to do TDD (Test driven development) on repositories and viewmodels in Android. He uses Spek to demonstrate live how test driven development is done. The live demo lasted for over an hour.

That’s all folks! Stay tuned for more we have exciting meetups in plan for July.

About the author

Sid Patil is an Android Engineer and Kotlin Advocate based in Berlin, Germany. He works at Delivery Hero, building ordering experiences for food delivery and Q-commerce brands like Foodpanda and Foodora.
